WEB DESIGN HOMEWORK

A PLOT JOURNEY

OUR GOAL

In this project, my aim is to analyse weather patters based on the distance to the equator. To accomplish this analysis, we need to use Python script to visualize the weather of over 500 cities across the world of varying distance from the equator.


OUR PLAN

In order to accomplish the task, we must first assembled the dataset of more than 500 cities using the citipy python library and python to randomly generate coordinates to get the nearest city. Using MatPlotLib, we used each city's maximum temperature, humidity, cloudiness, or wind speed and plotted the data with the latitude on the appropriate graphs.
